EPISODE I NEW PROJECTS

Aqui estão alguns dos projetos que desenvolvi durante minha trajetória como profissional de tecnologia. Eles incluem trabalhos pessoais que executei tanto dentro quanto fora da Trybe. Entre esses projetos, destaco o Pixels-Art, uma paleta de cores que utiliza HTML, CSS e JavaScript, e o Carrinho de Compras, que utiliza HTML, JavaScript e API do Mercado Livre. Também trabalhei em projetos que envolvem consumo de API, organização de rotas e biblioteca do NPM, além de filtragens, principalmente com React. Esses projetos representam minha experiência em habilidades de Front-End, como Git, HTML, CSS, React, Redux, Context API e RTL, bem como habilidades de Back-End, incluindo Docker, MYSQL, MSC, TypeScript, POO, SOLID, MongoDB e NoSQL. Entre os projetos de Back-End, desenvolvi soluções que utilizam filtragens de dados e manipulação de tabelas, normalização de dados, organização e verificação com querys. Também desenvolvi uma API de um CRUD (Create, Read, Update e Delete) com arquitetura MSC (model-service-controller), e outros projetos que utilizam Node.js com o pacote Sequelize. Trabalhei em uma API usando TypeScript com chave JWT, aplicando princípios de SOLID e POO, método TDD e integração com o banco de dados MongoDB.
